# PowerBI_Sales_Dashboard
Interaction Power BI dashboard for sales analysis using DAX, Power Querry and Data Modeling # Analytical Sales Dashboard

This Power BI dashboard analyzes sales performance, profit trends, product categories, and regional insights.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Features
- KPI Cards
- Profit Analysis
- Sales Trends
- Category Insights
- Interactive Filters
