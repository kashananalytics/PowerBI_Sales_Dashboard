# Analytical Sales Dashboard | Power BI

This project is an interactive Sales Analytics Dashboard developed using Microsoft Power BI to analyze business performance and generate meaningful insights from sales data.

The dashboard focuses on tracking sales, profit, cost, product performance, and regional analysis through clean and interactive visualizations.

## Project Highlights
- Sales & Profit Analysis
- Brand-wise Performance Tracking
- Category-wise Quantity Analysis
- City-wise Sales Visualization
- KPI Monitoring
- Interactive Filtering & Navigation

## Tools & Technologies Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel / CSV Dataset

## Features
- Interactive KPI Cards
- Dynamic Charts & Visuals
- Geo Map Analysis
- Business Performance Insights
- Clean Dashboard Layout
